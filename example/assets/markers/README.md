# Marker Pattern Files

Place AR marker pattern files in this directory.

## Hiro Marker

The `hiro.patt` file is the default Hiro marker pattern used by ARToolKit / JSARToolKit5.

You can download it from the [ARToolKit pattern repository](https://github.com/nicofirst1/ARToolKit-Pattern) or generate it using the [AR.js marker training tool](https://ar-js-org.github.io/AR.js/three.js/examples/marker-training/examples/generator.html).

Print the Hiro marker image at roughly 8 cm × 8 cm for best results with the default `physicalWidth: 0.08` setting.
