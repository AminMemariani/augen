# Marker Pattern Files

Place your AR marker pattern files in this directory.

## Hiro Marker

The Hiro marker is the default test marker for AR.js / JSARToolKit5.

### How to obtain `hiro.patt`

1. Download from the AR.js repository:
   https://raw.githubusercontent.com/AR-js-org/AR.js/master/data/data/patt.hiro

2. Save it as `hiro.patt` in this directory.

### How to print the Hiro marker

1. Search for "Hiro AR marker" online or visit:
   https://chev.me/armarker/ (select "Hiro" preset)

2. Print the marker on white paper at a reasonable size (8–12 cm recommended).

3. Ensure the black border is clearly visible with good contrast.

### Tips

- Print on matte paper to reduce glare.
- Tape the marker to a flat, rigid surface for best tracking.
- Ensure good, even lighting — avoid harsh shadows on the marker.
